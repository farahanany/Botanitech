# **React Native App Using Flask API and Firebase**

This React Native application integrates a Flask API and Firebase to provide real-time monitoring, crop prediction, and device control for a smart farming system.

---

## **Features**

- **Navigation**: The app includes a `MainContainer` that handles navigation between multiple screens.
- **Crop Prediction**: The Flask API runs a downloaded model to predict crops based on input data.
- **Real-Time Monitoring**: Firebase is used for real-time data monitoring, displaying sensor data for each node in the app.
- **Actuator Control**: A control page allows users to turn the actuator (water pump) on/off directly from the app.
- **Image Capture**: A detect page enables users to use the phone's camera to capture images and upload them to Firebase.
- **Node Details**: Users can view data for individual nodes and their statuses.
